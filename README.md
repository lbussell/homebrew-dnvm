# dnvm Homebrew Tap

## How do I install dnvm via Homebrew?

`brew install lbussell/dnvm/dnvm`

Or `brew tap lbussell/dnvm` and then `brew install dnvm`.

Or, in a `brew bundle` `Brewfile`:

```ruby
tap "lbussell/dnvm"
brew "dnvm"
```
